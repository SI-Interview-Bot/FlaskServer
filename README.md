# Python Server

## Setting up for Linux

### Workspace:
1. Install `python3`:
    `sudo apt install python3`
2. Install `pip3`:
    `sudo apt install python3-pip`
3. Install `flask`:
    `pip3 install Flask`
    `sudo apt install python3-flask`
4. Install `curl`:
    `sudo apt install curl`
